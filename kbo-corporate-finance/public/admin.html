Recevoir les messages par e-mail

L'envoi se fait par API HTTP (Brevo) — port 443, jamais bloqué par Railway (contrairement au SMTP). Créez un compte gratuit sur brevo.com, vérifiez votre adresse d'expéditeur, générez une clé API, puis ajoutez ces variables d'environnement sur Railway :

  • BREVO_API_KEY = votre clé API Brevo
  • MAIL_FROM = votre adresse expéditrice vérifiée chez Brevo
  • (alternative) SENDGRID_API_KEY si vous préférez SendGrid

Vérification de l'état…

Les messages arriveront à l'adresse définie dans Réseaux & Contact → Adresse e-mail de réception. Sans configuration, les messages restent enregistrés mais ne sont pas envoyés.
